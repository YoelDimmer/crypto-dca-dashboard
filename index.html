📖 Cómo funciona el sistema
Reglas de compra (C Moderado)
El multiplicador se calcula sobre tu cost basis (precio promedio de compra) por cada asset:
Sin pérdida o en ganancia1.00x base
Pérdida -10% a -25%1.50x base
Pérdida -25% a -40%2.00x base
Pérdida más de -40%2.50x base
Tipos de transacción
📥 Compra: registrás compra real de cripto
📤 Venta: vendés cripto, recibís USDT
💵 Aporte USDT: agregás USDT al pool (esperando comprar)
🚪 Retiro USDT: sacás USDT del exchange
Cost basis: al vender, tu cost basis se mantiene (precio promedio de las compras restantes). Las ventas se acumulan como "USDT realizado".
